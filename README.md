# Moralis Test Task

## FAQ

#### How to run tests?

- Clone the repo
- Install dependecies ```npm i```
- Fill .env 

headless mode - ```npm run test```

ui mode - ```npm run test:ui```

#### How to generate allure report

You need to have installed allure locally and then run

```npm run allure:report```
